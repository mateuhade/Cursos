Algorítmos: são um conjunto de etapas que devem ser seguidas para resolver um problema.
Automação:  Automação é o uso de máquinas para realizar algorítmos de forma automática ou semi-automática
Programa:   Programas são conjuntos de algorítmos que são automatizados por um computador



----Como fazer um programa de computador----
é necessário:
1. Uma linguagem de programação                  (regras léxicas e sintáticas para escrever um programa, como Python, Java, C++)
2. Um IDE/Ambiente de desenvolvimento integrado  (Software usado para editar e testar um programa)
3. Um compilador                                 (transforma o código fonte em código objeto)
4. Um gerador de código ou máquina virtual       (Permite a execução do programa)

- regra léxica: ortografia                     (Cachorro está certo, Caxorro está errado; print() está certo, primt() está errado)
- regra sintática: gramática                   (O cachorro comeu está certo, A cachorro comeu está errado; x=2+y está certo, x=+2y está errado)
.                                              (Syntax error e Lexical error)

- IDE: softwares usados para construir programas(C e C++ usam Code Blocks; Java usa Eclipse ou NetBeans; VSCode)

- Código fonte:                                 (é aquele código que é feito pelo programador em uma linguagem de programação)
- Compilador:                                   (transforma o código fonte em código objeto, que pode ser utilizado pela máquina automatizadora)

Relação entre linguagens e sistemas operacionais:
    C/C++: o código fonte é compilado em um código objeto e gera um código executável que vai ser diferente para cada sistema operacional,
necessitando de uma recompilação completa para cada sistema operacional.
    PHP, Python, JavaScript: o código fonte é compilado em um código objeto e enviado para um interpretador específico para cada sistema
operacional, ou seja para trocar o sistema operacional aqui, apenas é necessário trocar o interpretador.
    Java, C#, o código fonte é precompilado em um arquivo chamado Bytecode que é executado por uma máquina virtual específica para cada sistema
operacional, ou seja para trocar o sistema operacional aqui, apenas é necessário trocar a máquina virtual/gerador de código.
